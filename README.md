# AI 별자리 복습 챗봇 v3

## 기능
- 별자리 Q&A (계절·월·방위, 한국 기준)
- 밝은 별 → 별자리 질의응답
- 랜덤 4지선다 퀴즈 + 채점
- 정답을 숫자(1~4)만 입력해도 처리
- OpenAI 키 없을 때도 오프라인 동작

## 실행 방법

### 1) Node.js 로컬 실행
```bash
# Node.js 18+ 설치 필요
cd ai-constellation-bot-v3
npm install
cp .env.example .env    # 필요시 API 키 넣기
npm start
